## 主题配色

通过简单的配置一些主题配色变量，带调整应用的色彩外观。

这个是主题配色文件夹，不要搞错了哟


## 简介

主题配色用于进行全局色彩调配和提供一套色彩方案。  
遵循约定还可以通过色彩为UI提供一定的层级结构体现，语义化体现。

主题配色体系可以让你专注于进行全局的色彩调配，
并以最优的速度进行加载应用，并可与不同的主题风格进行搭配，创造独一无二的体验！

欢迎定制和分享贡献你的主题配色！
