# 主题风格

进行主题超越配色外的风格和功能美化。

这个文件夹下是主题风格文件夹，不要搞错了哟


## 简介

可以用自定义css和注入JavaScript充分利用资源文件等的方式来进行主题美化，更接近真正意义上的主题

为了丰富主题的功能，主题风格除了支持自定义css进行样式美化外

UI系统还为主题提供了可选的JavaScript辅助支持，并提供部分API供开发者调用。

欢迎定制和分享贡献你的主题风格！
