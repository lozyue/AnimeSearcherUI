# Fantasy

磨砂风梦幻主题风格 Fantasy.


## Features

### 自定义背景

背景使用高斯模糊默认随机次元风API。

并支持背景选项，可自定义高斯模糊度以及背景图像或API的链接。


同时为AuiPlayer播放器增加了背景图。


### 播放器增强

- 播放器快捷键

基于番剧片头片尾时长严格正态集中于 01:30s 的统计规律，
增加按A快速空降前进 90s 支持。

- 外部弹幕支持

启用外部弹幕支持主题选项后将在 AuiPlayer 页面与 Anime分区播放页面
增加一个上传本地弹幕文件和通过URL导入弹幕控制组件。并提供清除弹幕池按钮。

目前支持XML格式的外部弹幕。


### 一言扩充

增加了一些美词美句.


## 关联主题配色

亮色模式下启用 `dew`

暗色模式下自动启用 `dew-dark`
